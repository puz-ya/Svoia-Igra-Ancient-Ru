web_svoia_igra
==============

Quiz about ancient Russia

This is simple HTML + JavaScript site.
It contains +/- 30 questions about ancient Russia.
It can be usefull for history teachers and self-education.

To begin: just open index.html with your browser (IE, Firefox, Opera, Chrome, other).
To unhide questions, that has been already viewed, just press F5 (Refresh page).

There are no answers (in this version) and some questions are hard to answer :D
Have a nice test!
